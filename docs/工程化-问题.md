### 问题 1

chalk@5.0 版本不支持 node 环境中的 require 导入

<alert>解决方案</alert>

- chalk 降级至 v4.0 版本
- 修改 package.json 中的 `type=module`


### 问题 2

```
Error: spawn npm  ENOENT
```

在windows下npm的执行命令不同

<alert>解决方案</alert>

做判断
```
await spawn(process.platform === 'win32' ? 'npm.cmd' : 'npm', ['install'], { cwd: `./${name}` })
```
